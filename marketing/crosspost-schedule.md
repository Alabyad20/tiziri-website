## How to batch-schedule (one sitting)

**Facebook** — business.facebook.com → **Planner** → Create post → Reel → upload the file, paste the caption (same as Instagram, from this table's caption column), click the arrow next to Publish → **Schedule** → set the date at **18:30**. Repeat down the table.

**YouTube** — studio.youtube.com → Create → Upload videos (you can drag several at once) → for each: paste title/description/tags from marketing/youtube-launch-pack.md (entry number in the table) → Visibility step → **Schedule** → set the date at **18:30**. 

⚠ **One-of-one rule:** if a rug SELLS before its scheduled date, delete its pending scheduled posts on Facebook AND YouTube (and the IG calendar entry) the same day you delist it.
